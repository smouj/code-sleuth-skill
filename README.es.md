Code Sleuth skill for OpenClaw.

## Qué Soluciona
Analiza repositorios de código de forma inteligente para detectar patrones, antipatrones y oportunidades de mejora.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales, listos para producción
- Enfoque de seguridad primero
- Soporte para reversión

## Uso
/code-sleuth

## Ejemplo
```
Analizar el código en busca de problemas de programación.
```

## Instalación
Esta habilidad está disponible automáticamente en OpenClaw.
